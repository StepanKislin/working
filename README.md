# Working — Платформа для удалённой работы и бизнеса

![Working Logo](assets/images/icon.png)


**Working** — это современная веб-платформа, созданная для тех, кто ищет удалённую работу, хочет предложить свои услуги или найти исполнителей для проектов. Независимо от вашего опыта, локации или формата занятости — здесь вы найдёте то, что вдохновляет.

> 💡 **Бизнес или карьера в надёжных руках**

---

## 🚀 Основные возможности

- 🔍 Поиск вакансий и услуг по всему миру
- ✍️ Публикация собственных вакансий, услуг и заявок
- 🌐 Адаптивный интерфейс под любые устройства
- 📊 Аналитика откликов и рейтинги пользователей
- 🛡️ Безопасная среда с поддержкой 24/7
- 🆓 Бесплатная регистрация и базовый функционал

---

## 🛠️ Технологии

- **Frontend**: HTML5, CSS3, JavaScript (без фреймворков)
- **Backend**: PHP 8+
- **База данных**: MySQL (через PDO)
- **Сессии**: Управление авторизацией через `$_SESSION`
- **Безопасность**: Экранирование вывода (`htmlspecialchars`), подготовленные запросы.
